This repository contains the `.sty` file for the StarLab beamer theme, as well as an example set of beamer slides using this `.sty` file. 

<img src="https://github.com/paynesa/starlab-beamer/blob/main/figs/title.png?raw=true" alt="I <3 StarLab" width="500"/>

The relevant files are as follows:
- `starlabtheme.sty` defines the style of the StarLab beamer slides. 
- `main.tex` is an example LaTex file using the `starlabtheme` style. It should be compiled with `pdflatexmk`.
- `custom.bib` is the custom bibliography file for you to add your own BibTex entries to.
- `acl_natbib.bst` is the ACL bibliography formatting file, which I used to format the bibliography in the starlabtheme slides.
- `figs` contains the relevant logos for StarLab @ SBU. StarLab @ CUNY is welcome to add theirs as well.
